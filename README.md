# SCS

SCS(Study Computer Science)는 컴퓨터 과학 전공 지식을 학습하기 위한 웹 서비스이다.

**Commit Message Rule**
- 초기 커밋: Init
- 기능 추가: Feat
- 버그 수정: Fix
- 코드 리팩토링: Refactor
- 문서 업데이트: Docs
- 성능 개선: Perf
- 스타일 변경: Style
- 코드 제거: Remove
- 테스트 추가: Test
- 의존성 업데이트: Deps
- 데이터베이스 마이그레이션: DB
- 오타 수정: Typo
